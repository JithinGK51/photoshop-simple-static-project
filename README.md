# LuxePhoto - Luxury Photography Studio

A professional, luxury-themed photography studio website designed with a focus on cinematic aesthetics and fluid user experience.

## 🚀 Live Demo
[View Live Project](https://jithingk51.github.io/photoshop-simple-static-project/)

## ✨ Key Features
- **Cinematic Design**: High-end typography and immersive visual layout.
- **Dynamic Interactions**: Custom-built cursor with particle effects.
- **Theme Support**: Seamless transition between Light and Dark modes.
- **Interactive Gallery**: Sliding showcase of professional photography categories.
- **Responsive Layout**: Optimized for all device sizes.
- **Asset Integration**: Smooth animations and neural-style backgrounds.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Advanced layouts, custom properties, and animations.
- **JavaScript (Vanilla)**: Theme toggling, scroll animations, and interactive elements.

## 📂 Project Structure
- `index.html`: Main landing page architecture.
- `contact.html`: Specialized contact interface.
- `footer.html`: Reusable footer component.
- `styles.css`: Core design system and visual styles.
- `script.js`: Interactive logic and animation handlers.
